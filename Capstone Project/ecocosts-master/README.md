# EcoCosts
A budgeting application for Software Engineering II.

## Layout
- Folder `build` holds executables and template files necessary to run the
  program. This folder appears after the build script is ran.
- Folder `templates` holds the template files necessary for the front-end to
  incorporate the back-end.
- Folder `database` holds all necessary files to create and populate the
  database with test data.

## Install

### Linux
1. Install PostgreSQL, Golang from your distribution package repository.
2. Run the setup `ecocosts.sh`.

### Windows
1. Install [Golang](https://golang.org/dl/)
2. Install [PostgreSQL](https://postgresql.org/download/windows)
